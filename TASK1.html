What is Browser

A browser, short for web browser, is a software application used to access and view websites and other content on the internet.

Eg: Chrome,FireFoxx

What is Internet

The Internet is a global network of interconnected computers and devices that communicate using standardized protocols, allowing people to send, receive, and access data from anywhere in the world.

Eg:Google Search,Youtube

What is HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and structure web pages on the Internet.

Difference between html and Programming language

HTML is a markup language used to structure and display content on web pages. A programming language (like JavaScript or Python) is used to perform logic, calculations, and control behavior.

What is URL

A URL (Uniform Resource Locator) is the address used to access a specific resource (like a webpage, image, or file) on the internet.

Eg:-https://www.google.com
